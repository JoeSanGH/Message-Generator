# Message Generator
**Description**
Every time the program is run, a randomized quote is output. 

## Project Objectives
- Build a message generator program using JavaScript
- Use Git version control
- Use command line
- Develop locally on own computer

## Prerequisites
- JavaScript
- Git & GitHub
- Command Line

## Usage
Run program to generate a new quote.
